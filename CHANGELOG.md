# Changelog

All notable changes to ClipMemory will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.1.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.1.0] - 2026-04-14

### Changed
- Updated app icon to new portal-ring design

## [1.0.0] - 2026-04-14

### Added
- Clipboard history with text, image, file, and rich content support
- Strip card UI with visual previews
- Pin and favorites for quick access
- Tag system with custom tagging
- Full-text search across clipboard history
- Drag and drop clips to other applications
- Multi-format clipboard support (text, RTF, HTML, images, file lists)
- OCR text extraction from images
- Paste stack for sequential pasting
- Rules engine for auto-tag and auto-pin
- Global keyboard shortcuts
- System tray integration
- Auto-update via Velopack
- Settings with General, Shortcuts, Capture, History, and Excluded Apps tabs
- Dark theme UI
- Localization support (English, Korean)
