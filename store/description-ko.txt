ClipMemory — Windows 11을 위한 모던 클립보드 관리자

간략 설명 (최대 100자):
복사하는 모든 것을 저장하는 경량 클립보드 관리자 — 텍스트, 이미지, 파일 등을 즉시 재사용하세요.

상세 설명:

ClipMemory는 시스템 트레이에 상주하는 모던 클립보드 관리자입니다. 복사하는 모든 것을 자동으로 저장하고, 클립보드 기록을 즉시 검색하고 재사용할 수 있습니다.

주요 기능

* 클립보드 기록 — 텍스트, 이미지, 파일, 리치 콘텐츠 자동 저장
* 스트립 카드 UI — 빠른 탐색과 미리보기를 위한 비주얼 카드 인터페이스
* 고정 & 즐겨찾기 — 중요한 항목을 항상 접근 가능하게
* 태그 & 검색 — 태그와 전문 검색으로 클립 정리 및 검색
* 드래그 앤 드롭 — 클립을 다른 앱으로 바로 드래그
* 다중 포맷 지원 — 텍스트, RTF, HTML, 이미지, 파일 목록 등
* OCR — Windows 기본 OCR을 활용한 이미지 텍스트 추출
* 페이스트 스택 — 여러 항목을 큐에 넣고 순서대로 붙여넣기
* 규칙 엔진 — 사용자 정의 규칙으로 자동 태그/자동 고정
* 키보드 단축키 — 빠른 접근을 위한 글로벌 핫키
* 자동 업데이트 — 백그라운드 자동 업데이트

특장점

* WinUI 3으로 제작된 Windows 11 네이티브 UI
* 모던 반투명 효과의 다크 테마
* 데이터 수집 없음 — 모든 데이터는 기기에만 저장
* 경량 — 최소한의 CPU 및 메모리 사용
* 설치 및 실행에 관리자 권한 불필요

시스템 요구사항

* Windows 11 22H2 이상
* x64 프로세서
